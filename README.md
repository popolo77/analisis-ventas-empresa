# Análisis de Ventas de una Pequeña Empresa

## Descripción del Proyecto

Este proyecto fue desarrollado como parte del Trabajo Práctico de Organización Empresarial de la Tecnicatura Universitaria en Programación (UTN).

El objetivo del proyecto es analizar datos de ventas comerciales utilizando Python, Git, GitHub, Jira y Google Colab, aplicando conceptos de trabajo colaborativo, control de versiones y trazabilidad.

## Integrante

* Mariano Popolo

## Escenario Seleccionado

Escenario B – Análisis de Ventas de una Pequeña Empresa.

## Tecnologías Utilizadas

* Python
* Pandas
* Matplotlib
* Git
* GitHub
* Jira
* Google Colab

## Estructura del Proyecto

* `/datos` → datasets CSV utilizados en el análisis.
* `/scripts` → scripts Python del proyecto.
* `/resultados` → gráficos y resultados generados.
* `README.md` → documentación principal del proyecto.

## Funcionalidades Implementadas

* Lectura de dataset CSV.
* Cálculo de ventas totales.
* Identificación del producto más vendido.
* Análisis de ventas mensuales.
* Generación automática de gráficos.
* Trabajo colaborativo mediante branches y Pull Requests.

## Instrucciones de Ejecución

1. Clonar el repositorio.
2. Abrir Google Colab.
3. Ejecutar el script `analisis_ventas.py`.
4. Verificar resultados en la carpeta `/resultados`.

## Buenas Prácticas Aplicadas

* Uso de control de versiones con Git.
* Trazabilidad entre Jira y GitHub.
* Uso de ramas para desarrollo.
* Protección de archivos innecesarios mediante `.gitignore`.
* Organización estructurada del proyecto.
